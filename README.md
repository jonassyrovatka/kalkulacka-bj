Kalkulačka Bankovních Jednotek (BJ)

Tato aplikace slouží pro automatický a přesný výpočet Získatelských a Kariérových bankovních jednotek pro finanční poradce napříč různými institucemi a typy produktů.

Podporované moduly

Kalkulačka obsahuje tři oddělené a samostatně fungující moduly:

Investice: Pravidelné i jednorázové investice, řešení vstupních poplatků, omezení délky investičních horizontů a inteligentní matice koeficientů (včetně Amundi, Conseq, Partners Invest atd.).

Životní pojištění: Podpora režimů PWL, UpFront PWL a Semi PAYG. Aplikace automaticky odlišuje malá a velká rizika a chytře odděluje rizikovou a investiční složku u produktů typu Flexi, Šťastný život a dalších.

Úvěry: Spotřebitelské, hypoteční, firemní i stavební úvěry. Aplikuje specifické výjimky metodik (např. fixní paušální odměny u Buřinky nebo výpočet z poplatku u RSTS).

Jak aplikaci spustit

Aplikace je napsána v čistém HTML, CSS (Tailwind) a JavaScriptu. Nemá žádné backendové závislosti.

Pro spuštění stačí dvakrát kliknout na stažený soubor index.html. Otevře se ve vašem výchozím webovém prohlížeči.

Nasazení přes GitHub Pages (Zdarma)

Pokud chcete kalkulačku sdílet s kolegy jako běžnou webovou stránku:

Vytvořte nový repozitář na GitHubu.

Nahrajte do něj soubory index.html a README.md.

V repozitáři přejděte do záložky Settings -> Pages.

V sekci "Build and deployment" (pod "Source") vyberte větev main a uložte.

GitHub vám vygeneruje veřejný odkaz (např. https://[vase-jmeno].github.io/[nazev-repozitare]/), na kterém aplikace okamžitě online poběží.
